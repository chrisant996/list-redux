
GENERAL KEYS:

           ESC  Exit the file chooser (and exits the LIST program).

       F1 or ?  Show this help.
           F12  Show original screen until a key is pressed.

      HOME/END  Move selection to the first/last file.
    LEFT/RIGHT  Move selection left/right one file.
       UP/DOWN  Move selection up/down one file.
     PGUP/PGDN  Move selection up/down one page of files.

         ENTER  View the selected file, or navigate to directory.
        L or V  View the tagged files.

        S or \  Search for text and tag files (any case).
  Shift-S or /  Search for text and tag files (case sensitive).
    Ctrl-Break  Cancel search.

             1  Show only file names.
             2  Show file names and sizes.
             3  Show file names, dates, and sizes.
             4  Show file names, dates, sizes, and attributes.

ACTION KEYS:

             .  Navigate to parent directory.
       F2 or '  Incremental file name search, jump to selected file.
       F5 or *  Refresh list of files.
             P  New path or file mask.

             A  Change file or directory attributes.
             N  Create new directory.
             R  Rename file or directory.
             W  Sweep files: run a program on the selected or tagged files.

           DEL  Delete the selected or tagged files.

             E  Edit the file with %EDTIOR% or notepad.exe.
         Alt-R  Run the file or program associated with the file.

TAGGING KEYS:

             T  Tag the selected file.
             U  Untag the selected file.
         SPACE  Toggle tagged for the selected file.
        Ctrl-A  Tag all files.
        Ctrl-N  Invert tagged files.
        Ctrl-T  Tag all files.
        Ctrl-U  Untag all files.
